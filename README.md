# Financial Risk Data Warehouse

## Project Overview

This project simulates a banking data warehouse built entirely with SQL.

The objective is to transform raw banking data into a structured analytical database that supports:

- Customer risk profiling
- Fraud detection
- Loan portfolio analysis
- Transaction analytics
- Business intelligence reporting
